TikTok Publisher

A personal posting tool for accounts owned by Anthony Calise.

What it does

TikTok Publisher is a private automation script that publishes pre-generated slideshow posts (images plus captions) to a small set of TikTok accounts that I personally own and operate. It is not a multi-tenant product, not offered to other creators, and not available for download or sign-up.

Accounts published to

  • @yourgutbestie — gut health educational content
  • @fuelmodemedia — nutrition educational content
  • @testosterone.score — men's health educational content
  • @mayaoutside — lifestyle content

Each of these accounts is owned by me. I authorize the integration once per account using the standard TikTok OAuth flow.

How it works

  • A local script generates a 5–10 image carousel and a caption for an account.
  • The images are uploaded to a verified URL prefix on this domain.
  • The script calls the TikTok Content Posting API (Direct Post, photo) to publish the carousel to the authorized account.
  • Once TikTok confirms the upload, the source images are removed from this domain.

Why I built it

The accounts post on a daily cadence. Manually exporting images, switching accounts in the TikTok app, and pasting captions multiple times a day is the bottleneck I am replacing. The integration is a personal time-saver, not a product.

Scopes used

  • video.publish — required to publish photo carousels via Direct Post.

No other scopes are requested. The integration does not read user data, comments, analytics, follower lists, or messages.
