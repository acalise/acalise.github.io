Acalise Publisher Privacy Policy

Last updated: May 21, 2026

Overview

Acalise Publisher ("the integration") is a publishing tool that posts photo carousels and short videos to TikTok via the Content Posting API. This policy describes what data the integration accesses, sends, and stores.

Data accessed from TikTok

The integration accesses only what is required to publish a post via the Content Posting API:

  • OAuth access token, refresh token, and open ID — issued by TikTok when an account authorizes the integration. Used to authenticate posting requests.
  • Granted scopes — the integration requests video.publish only.

The integration does not read videos, comments, follower lists, direct messages, analytics, profile data, or any other TikTok content. It does not call any read endpoint.

Data sent to TikTok

To publish a post, the integration sends to TikTok:

  • A set of image URLs (for a photo carousel) or a single video URL (for a short video), all hosted on a verified prefix of acalise.com.
  • A caption, hashtags, and post-visibility settings.
  • The OAuth access token for the target account.

Data storage

OAuth tokens are stored locally and are never transmitted to a third party. Photos and videos served from acalise.com are public during the upload window because the TikTok server requires HTTPS GET access without authentication.

Sharing of data

The integration does not share, sell, lease, or transfer any TikTok-related data to any third party. The only recipient of TikTok-related data is TikTok itself, via the official Content Posting API.

Data retention

  • OAuth tokens: retained until access is revoked in TikTok settings or the integration is rotated.
  • Post metadata (timestamps, post IDs returned by TikTok): retained locally for scheduling and deduplication.

Revoking access

Access can be revoked at any time by visiting TikTok → Settings and Privacy → Security and Permissions → Manage app permissions, and removing Acalise Publisher.

Children

The integration is not directed to anyone under 13 and does not collect data from children.
