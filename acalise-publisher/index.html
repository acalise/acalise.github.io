Acalise Publisher

A publishing tool for TikTok photo carousels and short videos.

What it does

Acalise Publisher posts photo carousels and short videos to TikTok via the Direct Post API. Authorized accounts grant the tool publish access once, and the tool handles the upload-and-publish step on their behalf.

How it works

  • An external pipeline produces a 5–10 image carousel or a 1080×1920 mp4, plus a caption.
  • The media is uploaded to a verified URL prefix on acalise.com.
  • The authorized account is identified via TikTok’s OAuth flow (Login Kit).
  • The tool calls the TikTok Content Posting API (Direct Post, PULL_FROM_URL) with the media URLs and caption.
  • Publish status is polled until PUBLISH_COMPLETE.

Scopes used

  • video.publish — required to publish photos and videos via Direct Post. This is the only scope the integration actively calls.
  • user.info.basic — automatically included by Login Kit because the OAuth flow runs through it. Only the user’s open_id is stored.
  • video.upload — automatically included with the Content Posting API product. Not called by this integration.

The integration does not read user data, comments, analytics, follower lists, or messages. No read endpoints are accessed.
