What's included

Everything you need. Nothing you don't.

🎨
HeroUI Native
Full component library — Button, Card, Input, Modal, Select, and 15+ more, all pre-wired.
🌗
Dark & Light Mode
System-aware with a manual toggle. Persisted to device. Zero flash on launch.
🎯
One-line Accent Color
Change ACCENT_COLOR in one file. Both modes update across the entire app instantly.
📱
Tab Navigation
4 pre-configured tabs with custom SVG icons. No icon font dependency.
🚀
Onboarding Flow
3-step onboarding with spring-animated step indicator. Skips on subsequent launches.
📦
AsyncStorage
Typed wrapper with get, set, remove, has, keys, and clear.
🔔
Notifications
Local notifications helper with daily reminders, one-off scheduling, and permission handling.
📳
Haptics
Semantic haptic feedback — light, medium, success, error, and more.
💰
AdMob (optional)
Banner + interstitial, disabled by default. Enable with a single env variable. Test IDs pre-configured.
🟦
TypeScript Strict
Strict mode throughout with @/ path aliases configured out of the box.
💨
NativeWind v4
Tailwind CSS in React Native, tree-shakeable at build time.
📐
expo-router v4
File-based routing with type-safe links. Add screens by adding files.
Get started

Up and running in minutes.

1
Clone the repo
git clone https://github.com/acalise/expo-heroui-starter.git
cd expo-heroui-starter
2
Install dependencies
npm install
3
Set up environment
cp .env.example .env
4
Start the dev server
npx expo start

Scan the QR code with Expo Go, or press i / a for simulator.

5
Change your accent color
// constants/theme.ts
export const ACCENT_COLOR = '#8b5cf6'; // ← change this

Both modes update across the entire app. That's it.
